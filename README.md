# Animated Loading Spinners

A collection of lightweight, CSS-only loading spinners for various use cases.

## Features

-   **CSS Only**: No JavaScript or images required.
-   **Lightweight**: Minimal code footprint.
-   **Variants**: Circle, Dual Ring, Dots, and Ripple effects.

## Installation

1.  **Clone the repository** OR copy the CSS.
2.  **HTML Usage**:
    ```html
    <link rel="stylesheet" href="css/spinners.css">
    ```

## Usage

```html
<div class="spinner-circle"></div>
<div class="spinner-dual-ring"></div>
<div class="spinner-dots"></div>
<div class="spinner-ripple"><div></div><div></div></div>
```
